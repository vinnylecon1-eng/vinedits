## 🚀 Quick Start

### Prerequisites
- Node.js 18+
- npm or pnpm
- Social media accounts (Instagram, TikTok, YouTube, Facebook)
- OpenAI or Claude API key (for content generation)

### Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/advanced-viral-agent.git
cd advanced-viral-agent

# Install dependencies
npm install

# Create environment file
cp .env.example .env

# Add your API keys to .env
# - OPENAI_API_KEY or CLAUDE_API_KEY
# - INSTAGRAM_ACCESS_TOKEN
# - TIKTOK_ACCESS_TOKEN
# - YOUTUBE_API_KEY
# - FACEBOOK_ACCESS_TOKEN
# - SUPABASE_URL
# - SUPABASE_KEY

# Start development server
npm run dev
```

### First Steps
1. Visit `http://localhost:3000`
2. Create an account
3. Connect your social media accounts
4. Click "Analyze My Niche"
5. Wait 2-3 minutes for analysis to complete
6. Review your niche profile
7. Generate content
8. Schedule posts
9. Watch your engagement soar! 📈

## 🎯 How It Works

### Step 1: Connect Your Accounts
Connect your social media accounts using OAuth 2.0. Your credentials are encrypted and never stored.

### Step 2: Analyze Your Niche
The system analyzes your last 200 posts, followers, engagement patterns, and competitors to create a detailed niche profile.

### Step 3: Generate Content
AI generates 6 hooks, 5 titles, 3 descriptions, hashtag sets, and captions tailored to YOUR niche and audience.

### Step 4: Schedule Posts
System calculates optimal posting times and schedules content automatically.

### Step 5: Track Performance
Monitor engagement, reach, and growth in real-time. Get insights on what's working.

### Step 6: Optimize & Repeat
Use performance data to refine your strategy and generate even better content.

## 💡 Hook Archetypes Explained

### 1. **Fortuneteller** 🔮
Hints at transformation or result. Perfect for before/after content.
- Visual: Fast-paced opening, high contrast
- Text: "Here's what happened when..."
- Example: "I quit social media for 30 days"

### 2. **Experimenter** 🧪
Shows curiosity and relatability through testing.
- Visual: Experimentation setup
- Text: "I tested this for [TIME]"
- Example: "I posted 3 times daily for a week"

### 3. **Teacher** 📚
Promises fast, useful value.
- Visual: Clean, organized layout
- Text: "[NUMBER] proven strategies to..."
- Example: "3 ways to boost your engagement"

### 4. **Magician** ✨
Uses visual surprise and editing excellence.
- Visual: Snap transitions, outfit changes
- Text: "Watch this transformation"
- Example: "This editing trick gets millions of views"

### 5. **Investigator** 🔍
Teases hidden truth or insider knowledge.
- Visual: Mysterious, intriguing
- Text: "No one talks about this but..."
- Example: "This algorithm hack is what TikTok doesn't want you to know"

### 6. **Contrarian** 🔥
Flips popular belief with bold opinion.
- Visual: Bold, high-contrast
- Text: "Everything you know is wrong"
- Example: "Hashtags don't actually help your reach"
